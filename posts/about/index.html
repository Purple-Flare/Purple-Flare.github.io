Flare - What is it?

Why Flare?

With the dramatic increase in remote working and digital teams because of Covid-19 we are anticipating a surge in cyber threats throughout the world, specifically targeting . We wanted to create a solution to build a safer chat environment for any type of user.

How does it work?

Flare monitors channels and chat rooms from the background while only scanning messages that contain a URL. We compare the URL with security databases that are regularly updated and find any possible threat associated with it to let users know. Flare does NOT store any messages or URLs, instead we create a hash of the URL to help refer to it later on.
